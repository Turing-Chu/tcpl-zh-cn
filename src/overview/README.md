你可以从上到下阅读本章，但建议跳过这部分，因为一些相关概念无法单独解释。

在代码的例子中，注释 `#=>` 用于显示表达式的值。例如：

```crystal
a = 1 + 2
a #=> 3
```

注释里使用 `:` 用于显示变量的类型。

```crystal
s = "hello"
# s : String
```

让我们来从两个熟悉的例子开始学习这门语言。
